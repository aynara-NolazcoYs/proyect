# EduFuturo

Proyecto separado para un sistema de gestión similar al diseño mostrado en la imagen.

Estructura:
- `frontend/`: interfaz visual con dashboard, lista de estudiantes, lista de promotores y registro de matrícula.
- `backend/`: API REST separada para datos de estudiantes, promotores y matrículas.

Cómo usar:
1. Abrir `/Users/apple/EduFuturo/backend`.
2. Ejecutar `npm install`.
3. Ejecutar `npm start`.
4. Abrir `/Users/apple/EduFuturo/frontend/index.html` en el navegador.

> El frontend está preparado para consumir la API en `http://localhost:3000/api`.
